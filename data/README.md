# Data Folder

This folder stores:
- TradingView strategy exports (input)
- Converted trade files (intermediate)
- Cached option data (for performance)

## File Formats

### TradingView Export Format
Your Excel file should contain these columns:
- Trade #
- Type (Entry/Exit)
- Signal (Long/Short)
- Date
- Time
- Price

Place your TradingView exports here before processing.
